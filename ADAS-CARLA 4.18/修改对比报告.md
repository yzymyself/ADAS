### 修改1：CollisionSensor 碰撞回调增加 hud 空指针保护

**原始代码**（第105-106行）：
```python
actor_type = get_actor_display_name(event.other_actor)
self.hud.notification('Collision with %r' % actor_type)
```

**修改后**（第105-106行）：
```python
actor_type = get_actor_display_name(event.other_actor)
if self.hud is not None:  # [修改] hud 可能为 None（QtCarlaApp 模式下不使用 HUD）
    self.hud.notification('Collision with %r' % actor_type)
```

**原因**：QtCarlaApp 模式下传入 `hud=None`，直接调用 `self.hud.notification()` 会报 `AttributeError`。

## 三、新增文件 CameraController.py

独立摄像头视角控制器，替代 VehicleSpawner.py 中的 CameraManager 用于 QtCarlaApp 模式。
代码中所有新增内容均已标注 `[新增]` 注释。

核心类：
- **`ViewMode`** 枚举 `[新增]`：只保留 `DRIVER`（驾驶员）和 `TOP_DOWN`（俯视）两种视角，替代原始 6 个视角
- **`SpectatorCamera`** `[新增]`：单个 RGB 摄像头，attach 到车辆上自动跟随，使用 bounding_box 动态计算位置
  - 不依赖 HUD 对象，画面尺寸由参数直接传入
  - 提供 `set_offset()` 方法支持运行时切换视角参数
  - 使用线程锁保护帧缓冲区，避免回调与读取竞争
- **`CameraController`** `[新增]`：统一控制器，通过 `_View_PRESETS` 预设参数切换视角
  - 替代原始 CameraManager 通过数字 index 切换视角的方式
  - 提供 `set_mode()` / `get_image()` / `next_view()` 简洁接口
  - 被 QtCarlaApp 直接调用

关键设计：
- 摄像头 attach 模式（`Rigid` / `SpringArm`），不需要每帧 `set_transform`
- `get_image()` 返回 `(frame_id, numpy_rgb)`，Qt 端通过 frame_id 跳过重复帧
- yaw=0 朝前看（attach 模式下摄像头默认朝 +X 方向）
- 底部提供 pygame 版独立测试入口，不依赖 QtCarlaApp

## 四、新增文件 QtCarlaApp.py

用 PyQt5 替代 pygame 的独立显示界面，代码中所有新增内容均已标注 `[新增]` 注释。

核心类：
- **`QtCameraDisplay`** `[新增]`：QLabel 全屏显示摄像头画面，`setFocusPolicy(Qt.NoFocus)` 防止抢焦点
  - 替代原始 pygame surface 渲染方式
  - 通过 QImage → QPixmap → QLabel.setPixmap() 显示画面
- **`QtVehicleController`** `[新增]`：Qt 键盘事件驱动车辆控制，替代原始 pygame 版本的 KeyboardControl
  - 油门按住累加、松开清零，操控逻辑与原始 KeyboardControl 保持一致
- **`ConnectWidget`** `[新增]`：居中的"连接 CARLA"按钮界面，连接成功后自动隐藏
  - 原始 pygame 版本通过命令行直接连接，无 GUI 连接界面
- **`CarlaWindow`** `[新增]`：主窗口，管理连接、键盘事件、视角切换（V/C键）、帧渲染
  - 用 QTimer 替代 pygame 事件循环
  - 三个定时器：仿真 tick(50ms)、摄像头刷新(33ms)、状态打印(2s)
  - 不依赖 HUD 对象，World 初始化时传入 hud=None

关键修复（相对早期版本）：
- `AA_EnableHighDpiScaling` 在 `QApplication()` 构造前设置
- 摄像头用 `attach_to` + `SpringArm/Rigid` 而非每帧 `set_transform`
- 跳过重复帧渲染（`_last_frame_id` 检测）
- 主窗口定时抢回焦点（`_on_sim_tick` 中检测 `hasFocus()`）

CARLA 环境配置（自动检测路径）：
- 按优先级尝试：脚本相对路径 → 硬编码路径 → `CARLA_ROOT` 环境变量
- `_setup_dll_path()` 配置 Windows DLL 搜索路径
